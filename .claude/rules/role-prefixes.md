# Префиксы ролей (компакт)

> Source-of-truth: Pack (MIM.R.007, MIM.R.009). Каталог: `memory/roles.md`.

Доступные роли: **Навигатор** / **Диагност** / **Создатель паков** / **Оргразвитие** / **Консультант** (default).

Активация: начни сообщение с «Роль,» → полный контекст роли загрузится автоматически.
Пример: «Навигатор, с чего начать программу ЛР?»

Default без префикса: **Консультант** (отвечает на предметные вопросы через Pack/гайды).
